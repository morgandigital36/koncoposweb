Cara Penggunaan
1 Buka Google Sheet → Extensions → Apps Script
2 Hapus semua kode, paste isi file gas/Code.gs → Save
3 Pilih fungsi setupDatabase → klik ▶ Run → Allow
4 Deploy → New deployment → Web App → Execute as: Me → Access: Anyone
5 Copy URL deploy, paste di kolom URL di atas → Simpan
6 Tap Push ke Google Sheet untuk kirim semua data
